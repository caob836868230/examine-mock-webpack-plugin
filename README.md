# examine-mock-webpack-plugin
examine mock interface in your code

# usage
```javascript
  new ExamineMockyInterfacePlugin()
```

# params
`object`
|  param |  type  | remark |
|  ----  |  ----  | ----   |
| inlcude | String/Array/RegExp | 需要检查的模拟接口规则 |
| exclude | String/Array/RegExp | 排除检查的模拟接口规则 |
